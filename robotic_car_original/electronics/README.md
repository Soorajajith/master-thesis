Det här dokumentet beskriver vad som krävs för att kompilera och ladda in
koden på en Arduino Nano, eller en kompatibel enhet.

För att öppna filen krävs ett program med möjlighet att utveckla för Arduino,
till exempel den officiella mjukvaran på https://www.arduino.cc, eller
Atmel Studio med ett Arduino-plugin.

För att programmera Arduinon måste den vara inkopplad till datorn genom USB
och rätt serieport ska vara inställd i programvaran, och den kan sedan
programmeras genom ett knapptryck i mjukvaran.

På Linux ska datorn automatiskt hitta och installera drivrutiner för serie-
porten, och drivrutiner till Windows finns i filen arduino_driver.zip.
